Guida all'Installazione Locale

Requisiti Preliminari

Prima di iniziare, assicurati di avere installato sul tuo PC:

  • Node.js (Versione 18 o superiore): Scaricalo da nodejs.org. Scegli la versione LTS.

Installazione su Windows

  1. Scarica il codice: Esporta il progetto come ZIP da AI Studio o clona il repository.
  2. Prepara la cartella: Estrai i file nella cartella desiderata (es. Documenti/Planner).
  3. Apri il Terminale:
    • Vai nella cartella del progetto.
    • Fai Shift + Click destro in uno spazio vuoto della cartella e seleziona "Apri finestra PowerShell qui" o "Apri nel terminale".
  4. Installa le dipendenze (necessario solo la prima volta): 
    npm install

Avvio dell'Applicazione

Per avviare il planner, esegui il seguente comando nel terminale:

npm run dev

L'applicazione sarà accessibile nel tuo browser all'indirizzo:

http://localhost:3000
Nota sulla Porta: Se desideri usare la porta 3400, apri il file server.ts con un editor di testo e cambia la riga const PORT = 3000; con const PORT = 3400;.

Gestione dei Dati

Database Locale

I tuoi dati sono salvati nel file data.json nella cartella principale dell'app.

Backup & Portabilità

Copia il file data.json per i backup o l'intera cartella per spostare l'app su un altro PC.
